**Fecha de Entrega:** 27 de Diciembre de 2024

**Modulo:** Calidad 2.0 - Gestion de Talento Humano

---

## 1. DESCRIPCION DEL MODULO

El modulo **Calidad 2.0** es un sistema avanzado para la gestion del talento humano en instituciones de salud. Permite administrar todo el ciclo de vida del personal, desde su contratacion hasta su capacitacion continua, cumpliendo con las normativas colombianas de habilitacion.

Este modulo esta disenado especificamente para:
- Departamentos de Talento Humano
- Areas de Calidad
- Coordinadores de Capacitacion
- Administradores de la clinica

---

## 2. NORMATIVAS CUMPLIDAS

### 3.7 EVALUACIONES TIPO KAHOOT

Sistema interactivo de evaluaciones para medir el aprendizaje.

| Funcionalidad | Descripcion |
|---------------|-------------|
| Evaluacion pre-capacitacion | Mide conocimientos previos |
| Evaluacion post-capacitacion | Mide lo aprendido |
| Preguntas de opcion multiple | Facil respuesta |
| Tiempo por pregunta | Control del ritmo |
| Retroalimentacion inmediata | Respuesta correcta al momento |
| Calificacion automatica | Resultados al instante |
| Ranking de participantes | Competencia sana |
| Comparativo pre/post | Analisis del aprendizaje |

**Como funciona:**
1. El capacitador crea un banco de preguntas
2. Antes de la capacitacion, los participantes responden la evaluacion previa
3. Se realiza la capacitacion
4. Al finalizar, responden la evaluacion posterior
5. El sistema compara los resultados y muestra la mejora

## 5. FLUJOS DE TRABAJO

### 5.1 Flujo de Contratacion de Personal

```
1. Registro del empleado en el sistema
2. Carga de documentos requeridos
3. Verificacion de checklist segun cargo
4. Asignacion a proceso de induccion
5. Completar fases de induccion
6. Evaluacion de conocimientos
7. Generacion de certificado de induccion
8. Personal listo para laborar
```

### 5.2 Flujo de Capacitacion

```
1. Planificar capacitacion en cronograma
2. Crear sesion con fecha y hora
3. Crear evaluacion previa
4. Invitar participantes
5. Participantes completan evaluacion previa
6. Realizar la capacitacion
7. Tomar lista de asistencia
8. Aplicar evaluacion posterior
9. Comparar resultados pre/post
10. Generar certificados automaticamente
```

### 5.3 Flujo de Actualizacion de Documentos

```
1. Sistema alerta documento por vencer
2. Talento humano solicita actualizacion
3. Empleado entrega nuevo documento
4. Se carga al sistema
5. Se actualiza fecha de vencimiento
6. Checklist se actualiza automaticamente
```

## 8. BENEFICIOS DEL MODULO

1. **Centralizacion de informacion**: Todo el personal y sus documentos en un solo lugar

2. **Cumplimiento normativo**: Facilita el cumplimiento de las resoluciones de habilitacion

3. **Automatizacion**: Alertas, certificados y numeracion automatica

4. **Trazabilidad**: Historial completo de capacitaciones y documentos

5. **Ahorro de tiempo**: Procesos digitales en lugar de papel

6. **Medicion de resultados**: Evaluaciones comparativas del aprendizaje

7. **Preparacion para auditorias**: Documentacion lista para inspecciones
